# Fivezett Agents Skill Collection

個人のClaudeスキル集です。Vue / Tailwind開発で最適なパターンと実装ガイドを提供します。

## スキル一覧

### 1. VueUse Composables Reference
`@vueuse/core 14.3.x` の完全リファレンスガイド。DOM操作、ブラウザAPI、非同期状態管理などを標準コンポーザブルで実装する際のベストプラクティス。

- **対象**: `@vueuse/core` 14.3.0+
- **Vue**: 3.5.0+

**ディレクトリ**: `vueuse-composables/`

### 2. Tailwind CSS v4.1.x Reference
`Tailwind CSS v4.1.18` の完全リファレンスガイド。v3→v4移行時の落とし穴とv4固有の機能を網羅。

- **対象**: Tailwind v4.1.0+

**ディレクトリ**: `tailwind-v4/`
